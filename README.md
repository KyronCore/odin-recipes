# odin-recipes

## Introduction

This project is part of my early web development learning. It focuses on building a simple recipe website using HTML.

The goal is to practice structuring web pages using clean and semantic HTML.

## Skills Demonstrated

By completing this project, I will demonstrate:
- understanding of basic HTML structure
- use of headings, paragraphs, and lists
- linking between pages
- organizing content into multiple pages
- building simple static websites

## Self Reflection

This project helps me understand how websites are structured without using CSS or JavaScript. It improves my ability to think in terms of content structure and navigation.

It is a first step toward building more advanced and interactive web applications.
